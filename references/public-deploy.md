# 설문 공개 배포 (인터넷 어디서나 · LTE OK · 무료 · 새 계정 없음)

같은 WiFi가 없는 곳(행사장·길거리 등)에서 참여자가 각자 폰으로 응답하게 하려면 설문을
인터넷에 공개해야 한다. 가장 깔끔한 방법은 **Google Apps Script 웹 앱**이다:
구글이 설문 화면을 직접 호스팅하고, 응답은 **구글시트에 영구 저장**된다.
새 계정·서버·GitHub·요금 결제 모두 불필요(기존 구글 계정만 사용).

## 준비물
- 본인 설문의 `Code.gs` (설문 정의 CONFIG가 들어간 버전) + `Form.html`
  - 스킬이 생성해 주거나, `assets/apps-script/` 템플릿의 `CONFIG`를 해당 설문의 questions.json으로 교체.

## 단계 (구글 화면에서 진행 — 약 5분, 1회만)
1. **구글시트 새로 만들기** — sheets.new 접속 (응답이 여기 쌓인다)
2. 상단 메뉴 **확장 프로그램 → Apps Script** 클릭 → 스크립트 편집기가 열림
3. 기본 `Code.gs` 내용을 지우고 → 우리 **`Code.gs` 전체를 붙여넣기**
4. 편집기 왼쪽 **파일 + → HTML** → 이름을 정확히 **`Form`** 으로 만들고(확장자 .html 자동) →
   우리 **`Form.html` 전체를 붙여넣기**
5. 💾 저장(Ctrl+S)
6. 오른쪽 위 **배포 → 새 배포** → 톱니바퀴 ⚙ → **웹 앱** 선택
   - 설명: 아무거나 (예: survey v1)
   - **실행 계정: 나(본인)**
   - **액세스 권한: 모든 사용자** ← 이게 핵심 (로그인 없이 누구나 응답)
   - **배포** 클릭
7. 처음이면 **권한 승인** 창이 뜸 → 본인 구글 계정 선택 → "고급 → 안전하지 않음(이동)" →
   허용. (본인이 만든 스크립트라 정상)
8. 나온 **웹 앱 URL**(`https://script.google.com/macros/s/..../exec`)을 복사 → 이게 설문 공개 주소.

## QR 만들기
- 그 URL로 QR을 만들면 됨. 우리 `results.html`의 QR 칸에 이 URL을 붙여 넣어 생성하거나,
  아무 QR 생성 사이트 사용. → 이제 **LTE·어떤 망에서도** 폰으로 열린다.

## 결과 보기
- 응답은 구글시트에 실시간으로 쌓인다(구글 어디서든 확인).
- 우리 대시보드처럼 보고 싶으면: 구글시트 `파일 → 다운로드 → Excel(.xlsx)` →
  로컬 `import.html`에 드래그&드롭 → KPI·투표 집계·키워드까지 정리.

## 설문 내용을 바꾸려면
- Apps Script 편집기에서 `Code.gs`의 `CONFIG`를 수정 → 저장 → **배포 → 배포 관리 → 편집(연필) →
  새 버전 → 배포**. (URL은 그대로 유지됨)
- 헤더(열)가 바뀌면 기존 시트의 행과 어긋날 수 있으니, 큰 변경 시 시트의 헤더 행을 비우거나 새 시트를 쓴다.

## 주의
- 웹 앱 URL은 끝이 `/exec` 여야 실제 배포본. `/dev`는 본인만 보이는 테스트용.
- 액세스 권한을 "모든 사용자"로 안 하면 참여자에게 구글 로그인을 요구하게 되니 반드시 확인.
- 이 방식은 응답 저장이 구글시트라 **서버가 잠들어 데이터가 날아갈 걱정이 없다**(로컬 Node 방식의 한계 해결).
